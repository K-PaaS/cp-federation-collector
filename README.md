## 시작하기
federation-metric-api가 수행하는 애플리케이션 관리 작업은 다음과 같습니다.
- 멀티 클러스터의 Metric 정보 수집
- NATS에 Metric 정보 제공

#### federation-metric-api 빌드 방법
federation-metric-api 소스 코드를 활용하여 로컬 환경에서 빌드가 필요한 경우 다음 명령어를 입력합니다.
```
$ go build
```
